Die Regelungstechnik (RT) beschäftigt sich mit der selbsttätigen gezielten Beeinflussung des
Verhaltens von dynamischen Systemen. 

\begin{center}
\begin{tikzpicture}[node distance=2.5cm,auto,>=latex']
	\node [system] 		(system) 						{System};
	\node 			(in) 	[left of=system,  node distance=4cm, coordinate] {system};
	\node [coordinate] 	(out) 	[right of=system, node distance=4cm]		{};

	\path[->,thick] (in) edge node {Eingänge} (system);
	\draw[->,thick] (system) edge node {Ausgänge} (out);
\end{tikzpicture}

\end{center}


Die Eingänge werden in der RT aufgeteilt in von außen vorgebbare \underline{Eingangsgrößen} und in durch die Umgebung festgelegte, störend wirkende \underline{Störgrößen}. Von den Ausgängen werden nur diejenigen betrachtet, deren Verhalten unmittelbar interessiert. \underline{Ausgangsgrößen}.

\begin{center}
\begin{tikzpicture}[node distance=2.5cm,auto,>=latex']
	\node [system, pin={[arrowthick]above:Störgrößen}] 	(system) 						{dynamisches System};
	\node 							(in) 	[left of=system,  node distance=6cm, coordinate]{system};
	\node [coordinate] 					(out) 	[right of=system, node distance=6cm]		{};

	\path[->,thick] (in) edge node {Eingangsgrößen} (system);
	\draw[->,thick] (system) edge node {Ausgangsgrößen} (out);
\end{tikzpicture}
\end{center}

\underline{Gezielte Beeinflussung} heißt: Durch Vorgabe der Eingangsgrößenverläufe soll erreicht werden, dass die Ausgangsgrößen trotz Störeinwirkung ein gewünschtes \underline{Sollverhalten} aufweisen.

\section*{Allgemeine Aufgabenstellung der RT}
Entwurf und Bereitstellung einer Einrichtung, die - hinzugefügt zur Strecke - die Eingangsgrößen automatisch im gewünschten Sinne generiert. \underline{Selbsttätige gezielte Beeinflussung}.

\section*{Generelle Vorgehensweise zur Lösung}
\begin{enumerate}
	\item{mathematische Modellbildung der Strecke zur Abstraktion von deren physikalischen Ausprägung und Ermöglichung der Anwendung universell einsetzbarer, systemtheoretisch fundierter Vorgehensweisen in Schritt 2. und 3.}
	\item{Analyse des Streckenverhaltens}
	\item{Entwurf der Steuer- und Regeleinrichtung}
	\item{Realisierung der Steuer- und Regeleinrichtung}
	\item{Inbetriebnahme und Erprobung des Gesamtsystems}
\end{enumerate}
